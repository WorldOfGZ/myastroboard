## UpTonight Organization

When uptonight runs, all output files (PNG, JSON, TXT) are stored in catalogue-specific subdirectories:

```
uptonight/outputs/
├── Messier/
│   ├── uptonight-plot.png
│   ├── uptonight-report.json
│   ├── uptonight-bodies-report.json
│   ├── uptonight-comets-report.json
│   └── uptonight-moon.png
├── Herschel400/
│   ├── uptonight-plot.png
│   ├── uptonight-report.json
│   └── ...
├── OpenNGC/
│   ├── uptonight-plot.png
│   ├── uptonight-report.json
│   └── ...
└── ...
```

Config files for each dedicated catalogue are stored in folder `uptonight/configs`.
